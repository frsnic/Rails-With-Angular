# Rails with AngularJs
![Alt text](https://travis-ci.org/frsnic/rails-with-angular.svg "Optional Title")

### About
Angular CRUD with Rails Controller

***
### Useage
```
npm install -g bower
bower install
rake db:migrate
rake db:seed
```
